# Changelog

## Version 1.0.0

**Date:** 2026-06-04

### Added
- Added table-facing X/N/O safety tools with configurable anonymous presentation options.
- Added hand-raising tools: Cut in Line, Next In Line, and Back of the Line.
- Added presence tools: AFK and Open Door.
- Added participant/source tiles for visible safety, hand, and presence signals.
- Added owner/GM clearing behavior for active requests and status tiles.
- Added an editable Safety Reference drawer with inline title editing and inline reference box editing.
- Added Safety Reference box controls for adding, moving, duplicating, deleting, and saving boxes.
- Added client-side notification timer controls for collapsed-toolbar notifications.
- Added optional notification sounds with X Card priority sound behavior.
- Added CC0 Kenney Interface Sounds audio assets in OGG format.
- Added toolbar docking, resizing, viewport push-off, and client-local placement.
- Added Foundry settings for tool presentation, image preference, initial toolbar state, notification sounds, and notification volume.
- Added attribution and license documentation for bundled Kenney audio assets.
- Added All Rights Reserved license documentation for module code and original module content.

### Changed
- Initial public release.

### Fixed
- Initial public release.

### Known Issues
- Safety Reference boxes are plain multiline text only; markdown/rich formatting is not implemented.
- No chat output, macro slots, emergency pause, or Help Mode are included in this release.

### Migration Notes
- Initial public release.
- Existing pre-release worlds may keep local client settings and world settings, but this release is intended as the first public baseline.
